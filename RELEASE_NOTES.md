# CellAnalyzer - Release Package

## Содержимое релиза

Данный релизный пакет содержит все необходимое для запуска CellAnalyzer на другом компьютере.

### Файлы и папки

```
CellAnalyzer.exe          - Основной исполняемый файл
*.dll                     - Библиотеки Qt6 и OpenCV
platforms/                - Qt платформенные плагины
styles/                   - Qt стили
imageformats/             - Qt кодеки изображений
ml-data/                  - ML модели и скрипты
  ├── models/
  │   ├── yolov8s_cells_v1.0.onnx  - YOLO модель версия 1.0
  │   └── yolov8s_cells_v2.0.onnx  - YOLO модель версия 2.0
  ├── scripts/            - Python скрипты (опционально)
  └── docs/               - Документация по ML
settings.json             - Настройки приложения (создается автоматически)
README.md                 - Общая документация
CHANGELOG.md              - История изменений
```

## Системные требования

### Минимальные требования
- **ОС**: Windows 10/11 (64-bit)
- **RAM**: 4 GB
- **Процессор**: Intel Core i3 или аналог
- **Видеокарта**: Любая с поддержкой OpenGL 2.0+

### Рекомендуемые требования
- **ОС**: Windows 10/11 (64-bit)
- **RAM**: 8 GB или больше
- **Процессор**: Intel Core i5 или выше
- **Видеокарта**: NVIDIA GPU с CUDA (для ускорения, опционально)

## Установка и запуск

### Вариант 1: Простой запуск (рекомендуется)

1. Распакуйте весь архив в любую папку (например, `C:\CellAnalyzer`)
2. Запустите `CellAnalyzer.exe`
3. Приложение готово к работе!

**Важно**: Не перемещайте файлы из папки, все DLL и ml-data должны находиться рядом с .exe

### Вариант 2: Python не требуется

Приложение использует **ONNX Runtime** для запуска YOLO моделей, поэтому **Python НЕ требуется** для работы.

ONNX модели встроены в релиз и работают через OpenCV DNN модуль.

### Вариант 3: С Python (опционально, для разработчиков)

Если вы хотите использовать Python версию YOLO или модифицировать скрипты:

1. Установите Python 3.8+ (https://www.python.org/downloads/)
2. Установите зависимости:
```bash
pip install ultralytics opencv-python numpy torch
```

3. Python скрипты находятся в `ml-data/scripts/`

## Первый запуск

1. Запустите `CellAnalyzer.exe`
2. Нажмите **"+ Добавить изображения"** или перетащите изображения в окно
3. Нажмите **"Начать анализ (YOLO)"**
4. Дождитесь завершения анализа
5. Проверьте и исправьте результаты в окне верификации
6. Сохраните результаты в CSV

## Возможные проблемы

### "Не удается запустить программу, так как отсутствует MSVCP140.dll"

Установите **Microsoft Visual C++ Redistributable**:
- Скачайте с https://aka.ms/vs/17/release/vc_redist.x64.exe
- Установите и перезапустите приложение

### "Failed to load ONNX model"

Убедитесь, что:
1. Папка `ml-data/models/` находится рядом с CellAnalyzer.exe
2. Файлы `yolov8s_cells_v1.0.onnx` и `yolov8s_cells_v2.0.onnx` присутствуют
3. Файлы не повреждены (размер ~25MB каждый)

### "OpenCV error: DNN module is not available"

Проверьте наличие OpenCV DLL:
- `opencv_world4110.dll`
- `opencv_videoio_msmf4110_64.dll`
- `opencv_videoio_ffmpeg4110_64.dll`

## Портативность

Данный релиз является **полностью портативным**:
- Не требует установки
- Не изменяет системный реестр
- Все настройки сохраняются в `settings.json` рядом с .exe
- Можно запускать с USB флешки

## Обновление версии

Для обновления:
1. Сохраните `settings.json` (ваши настройки и коэффициент)
2. Удалите старую версию
3. Распакуйте новую версию
4. Скопируйте сохраненный `settings.json` в новую папку
